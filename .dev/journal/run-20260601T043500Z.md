## Notes

- The compaction summary message content has a "[compacted: N messages → M chars]\n"
  prefix prepended by the `Compactor`, so test assertions use `contains()` rather
  than exact match.
- `AgentEvent` dropped `Eq` because `serde_json::Value` (inside `ToolCall`) only
  implements `PartialEq`, not `Eq`. All existing uses of `assert_eq!` on `AgentEvent`
  only require `PartialEq`.
- `finalize_session_writer` correctly calls `Arc::into_inner(sw)` after `drop(runtime)`
  (which drops the `CompositeSink` → `SessionPersistenceSink` → cloned Arc), so the
  Arc reference count is 1 at finalize time.
